✅ 방법 1 — server.js 실행 (권장)
1. Node.js 설치 여부 확인: node --version
2. EuroJob.html 과 server.js를 같은 폴더에 두기
3. 터미널에서 실행:
node server.js
4. http://localhost:3000 에서 앱 열기 → 「🔍 유럽 공고 수집」
